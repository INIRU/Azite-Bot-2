## **About**

이 봇은 **아지트\<VALORANT\>** 서버의 **`자기소개` , `문의`, `모더`** 기능 처리의 목적으로 제작되었습니다.

## **Setup**

**[.env-template](.env-template)** 파일 이름을 **.env**로 수정하고 **.env**파일의 **`TOKEN`, `ClientId`, `GuildId` and etc..** 를 수정합니다.

**[setup.sql](setup.sql)** 을 실행하여 **DB Server Table**을 설정하여주세요.

```bash
npm install
npm start
# run setup.sql
```
